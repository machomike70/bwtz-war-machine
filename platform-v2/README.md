# Bear Witness War Machine Platform v2

This folder contains the new modular platform scaffold for the expanded **Bear Witness War Machine**.

The existing repository already contains the current Phase 1 Python-based daily growth engine. This v2 scaffold is being added beside it instead of overwriting it, so the live/working system stays intact while the larger platform is built module by module.

## Platform Rule

Bear Witness War Machine remains visibly branded as a Bear Witness product.

Licensed projects can configure their own:

- project profile
- token metadata
- social accounts
- campaign templates
- content presets
- brand voice
- assets
- analytics

But the platform identity remains:

**Bear Witness War Machine**  
**Powered by Bear Witness $BWTZ × Bitcoin Killas $BTCK**

## Build Direction

The goal is not to replace the current working War Machine immediately. The goal is to evolve it into a multi-project, multi-channel content and utility command center while preserving the existing repo history and operational code.
